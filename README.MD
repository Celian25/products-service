# Products Service
Сервис для вывода и добавления товаров по фильтрам. Планируется поставлять
в виде Telegram Web App внутри группы.

В дальнейшем планируется добавить интеграцию с телеграм группами для более эффективного управления контентом

Сервис будет состоять:
- Бекенд части - из него будут брать данные другие сервисы, а также на начальном этапе планируется работать с телеграмом.
- Админ часть - планируется поднять быстро на React Admin, для эффективного управления контентом
- Фронтенд часть - витрина товаров, именно это приложение будет в дальнейшем завернуто в Telegram Web App и будет доступно массовому пользователю

# Как развернуть

```bash
cd docker/local
```

```bash
docker compose up -d
```

После сборки можно перейти на localhost:8081 - там будет GraphiQL UI со всей схемой GraphQL API. 
Эндпоинт GraphQL - localhost:8081/graphql# products-service
